# Compiler From Scratch

This project implements an educational compiler developed entirely from scratch in C as part of a Computer Science course on compiler construction.

The compiler was incrementally developed throughout the semester, gradually adding support for lexical analysis, syntax analysis, semantic validation, and code generation for a simplified programming language.

The final version supports variables, control structures, functions, symbol tables, and generation of assembly code.

---

## Features

- Lexical analysis
- Syntax analysis (table-driven parser)
- Symbol table implementation
- Variable declarations
- Input / output commands
- Conditional statements (if / else)
- Loops (while)
- Function declarations
- Function calls
- Control flow jumps
- Intermediate / assembly code generation

---

## Language Features

The implemented language includes:

- variable declarations
- arithmetic expressions
- read / write commands
- conditional execution
- loops
- functions
- global and local scopes

---

## Final Assignment

The final stage of the project required implementing full support for functions including:

- program termination at the end of the main scope
- function code generation
- function call handling
- control flow jumps for function execution
- enforcement of function prototypes

--- 

## Compilation Process
 
Source Code  
↓  
Lexical Analysis (Flex)  
↓  
Syntax Analysis (Bison)  
↓  
Semantic Analysis  
↓  
Assembly Code Generation  

---

## Educational Context

This project was developed as part of the **Compiler Construction** course at the Federal University of Mato Grosso (UFMT).

The objective was to understand the internal architecture of compilers by implementing each stage manually.
